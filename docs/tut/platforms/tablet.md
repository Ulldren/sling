# Tablet

Sling supports tablets, adapting the interface dynamically based on the device's screen size and orientation.

## Layout

Depending on how the tablet is held and its screen resolution, Sling will automatically adjust the hotbar and inventory size:

* **Portrait Mode:** 3 hotbar slots and 4 inventory rows.
* **Landscape Mode (Small Screens):** 6 hotbar slots and 2 inventory rows.
* **Landscape Mode (Large Screens):** 10 hotbar slots and 4 inventory rows.
