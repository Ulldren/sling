# Why

Sling is a modern and reactive backpack UI designed for Roblox experiences. It replaces the default backpack system, giving you complete freedom to customize the interface however you want, dynamically categorize inventory items, and use or create custom themes.

![Sling Preview](/inventory-preview.png)

## What makes Sling different?

Leveraging the reactive library [Vide](https://github.com/centau/vide), sling have a comprehensible architecture, with separated components for clear responsibility, and highly readable codebase. Compatible with [UI Labs](https://ui-labs.luau.page/), allowing you to easily preview components.
