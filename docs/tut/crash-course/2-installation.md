# Getting Started

This goes over how to start setting up sling.

## Installation

:::tabs
==Wally
In your `wally.toml`, add the following line to your dependencies.

```toml
sling = "encodedlux/sling@0.1.3"
```

==Rbxm
1. Download the latest `sling.rbxm` model file from the [Github Releases](https://github.com/encodedlux/sling/releases/latest/) or by pressing the button below.
2. Open Roblox Studio and open your place.
3. In the **Explorer** window, insert the **Sling** model into **ReplicatedStorage**.

<Button href="https://github.com/encodedlux/sling/releases/latest/download/sling.rbxm" text="Download latest rbxm"></Button>

:::
